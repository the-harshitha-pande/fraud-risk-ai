# 💳 FraudRiskAI

An AI-powered system for detecting and managing financial fraud using explainable logic and data analytics.

## 🔗 Live Demo
🌐 https://fraud-risk-ai.vercel.app  
🤖 https://cddgetg-fraud-risk-ai.hf.space  

## 🚀 Overview
FraudRiskAI identifies risky transactions using a constraint-based approach and classifies them into:
- 🟢 Low Risk  
- 🟡 Medium Risk  
- 🔴 High Risk  

Based on classification, it provides actionable decisions such as **allow, verify, or block**.

## 🧠 Key Idea
Instead of relying purely on black-box models, this project uses **Constraint Satisfaction Problem (CSP) logic** to ensure:
- Transparency  
- Interpretability  
- Clear reasoning behind decisions  

## 📊 What It Covers
- AI-based fraud detection  
- Risk scoring and classification  
- Decision recommendations  
- Fraud-focused data analytics (dashboard, trends, segmentation)  

## 🛠 Tech Stack
- React.js  
- Python (CSP Logic)  
- Gradio  
- Vercel  

## ✨ Highlight
A complete end-to-end system combining **AI decision-making + data analysis** for fraud prevention.
